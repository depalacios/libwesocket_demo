# Websocket client connection using node
A simple websocket client connection using WS package and Node v24.11.1. 

## Install packages
Use one of the next commands:

```bash
npm i 
or 
npm install
```

## Run client
Update the next variable value with the corresponing server IP ADDR:

```bash
serverIp="YOUR_SERVER_IP"
```

Use the next command to run client connection:

```bash
npm start
```
